# Product Spec

## Problem Statement

Offshore fishers need a simple way to reason about pelagic habitat without pretending a model can know exact fish locations. This app turns oceanographic signals into transparent suitability scores.

## Target Users

Sydney and NSW offshore anglers, skippers, fishing clubs and future analysts exploring tuna, marlin, mahi mahi and offshore kingfish habitat.

## Why Offshore Pelagic First

Pelagic activity often relates to fronts, currents, temperature bands, shelf breaks, canyons and structure. Those features are a natural first fit for explainable habitat scoring.

## Target Region

Approximate v0.1 bounding box: north `-32.0`, south `-36.5`, west `150.5`, east `154.5`.

## Target Species

Yellowfin Tuna, Southern Bluefin Tuna, Striped Marlin, Blue Marlin, Mahi Mahi / Dolphinfish, Offshore Yellowtail Kingfish and Longtail Tuna.

## v0.1 User Journey

The user opens a local dashboard, selects a species, reviews mock ocean status, inspects a hotspot map, reads scored hotspot cards and checks the disclaimer before making any independent real-world decision.

## Main Screens

Single dashboard with header, species selector, layer controls, map placeholder, hotspot list, species profile, freshness notes and legal/safety disclaimer.

## Future Features

MapLibre layers, real SST, bathymetry, current/chlorophyll/SLA data, catch logs, ML baselines and forecast mode.
