# Project Plan

## Phase 1

v0.1 local mock demo with FastAPI, Next.js, synthetic ocean layers, POIs, rule-based species scoring, GeoJSON hotspots and tests.

## Phase 2

Real SST and bathymetry, MapLibre map, better POI layers and curated FAD points.

## Phase 3

Currents, chlorophyll, sea level anomaly, gradient computation and daily cached data.

## Phase 4

User catch log stored locally with catch/no-catch observations and private notes.

## Phase 5

Machine learning baseline using historical presence/catch logs and background samples.

## Phase 6

Species distribution modelling, spatiotemporal modelling and offshore plus estuary dashboard.
