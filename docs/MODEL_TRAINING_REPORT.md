## Limitations

- Occurrence data is presence-only.
- Background samples are pseudo-absence / available environment, not confirmed absence.
- There is no catch/no-catch effort dataset yet.
- The model target is relative habitat suitability, not exact fish location.
- Local Sydney performance depends on better local catch/tag data and reliable oceanographic feature extraction.
